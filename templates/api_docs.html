API Overview

The SkillSync AI Platform provides a comprehensive RESTful API for AI-powered career development and skill assessment. All endpoints return JSON responses and support CORS for web applications.

Authentication

Some endpoints require an xAI API key for AI functionality. Set your XAI_API_KEY environment variable for full features.

Quick Start Guide

Get started with the SkillSync API in minutes:

1. Test the Health Endpoint

curl https://skillsync-ai-platform-production.up.railway.app/health

2. Assess Your Skills

curl -X POST https://skillsync-ai-platform-production.up.railway.app/assess-skills \ -H "Content-Type: application/json" \ -d '{"skills_description": "Your skills here..."}'

3. Get Career Guidance

curl -X POST https://skillsync-ai-platform-production.up.railway.app/career-guidance \ -H "Content-Type: application/json" \ -d '{"current_skills": "Python, Django", "career_goals": "Data Scientist"}'
